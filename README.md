# Modulo - Education Reinvented
An education platform built to become the improved standardaized software used by students and educators.

## Overview
Modulo is a personalized education platform that aims to improve on the current method students access their course material, and assist educators in creating interactive material to keep students engaged outside and inside the classroom. Many students today access a site that is in all-in-one link holder and organizer to point and host powerpoints, links to PDFs, without actually creating a useful and engaging way to learn material and educators face this obstacle without any tools. This software will become an educators best friend, and a students lifeline for success.

## Features 
- User-Friendly Interface   
- Standardized Course Structures (i.e. Module overviews, universal lesson structures)
- Course Resource Accessibility
